Tunnel del vento 2D attorno a profili NACA

Simulazione qualitativa con metodo Lattice-Boltzmann (LBM): campo di velocità, vorticità, pressione, streamline di fumo e vettori portanza/resistenza al variare dell'angolo d'attacco. Non è una CFD validata — serve per intuizione didattica.

Basato su kutta (MIT) di Carlos Gimenes — portato nel browser via WebAssembly + Ebitengine.

Controlli tastiera

Clicca prima sul riquadro della simulazione per dare focus alla tastiera.

  • ↑↓ Angolo d'attacco
  • Tab Profilo NACA (ciclo preset)
  • V Campo: velocità / vorticità / pressione
  • S Linee di corrente (streamline)
  • G Effetto bloom sul fumo
  • [ ] Velocità di ingresso
  • Spazio Pausa / ripresa
  • R Reset del flusso
  • N Un passo (con pausa attiva)

Puoi anche digitare un codice NACA nel campo in alto (es. 2412, 0012) e usare gli slider in basso per angolo e velocità.

Nel browser non sono disponibili editor, apertura o salvataggio scene (E, O).
